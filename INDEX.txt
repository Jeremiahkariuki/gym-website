💡 QUICK REFERENCE
──────────────────────────────────────────────────────────────────────────────

TO RUN THE APPLICATION:
  1. cd c:\Users\Administrator\Desktop\gym
  2. env\Scripts\activate
  3. python manage.py runserver
  4. Visit http://localhost:8000/login/

TO CREATE ADMIN ACCOUNT:
  python manage.py createsuperuser

MAIN URL PATHS:
  /login/ ............................ Login page
  /logout/ ........................... Logout
  /dashboard/ ........................ Dashboard
  /members/ .......................... Member list
  /members/new/ ...................... Add member
  /members/<id>/ ..................... Member profile
  /plans/ ............................ Plan list
  /plans/new/ ........................ Add plan
  /attendance-report/ ................ Attendance stats
  /revenue-report/ ................... Revenue stats
  /admin/ ............................ Admin panel
